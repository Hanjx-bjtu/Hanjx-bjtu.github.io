# 🌐 量子计算概念可视化平台

本项目由Deepseek, Grok, Copilot辅助开发

## 📖 项目简介

本项目是一个基于Web的量子计算教学与演示平台，通过**三维可视化**与**交互式操作**直观展示量子比特的布洛赫球表示、多量子比特系统的纠缠特性以及常用量子算法的运行过程。平台集成了单比特/多比特可视化、量子门序列模拟、量子算法演示与测量模拟等功能，适用于量子计算初学者。

---

## 🧩 功能模块

### 1. **单量子比特可视化**
- 交互式布洛赫球，展示任意纯态在球面上的位置
- 支持常用基态（如 |0⟩、|1⟩、|+⟩、|-⟩ 等）一键设置
- 支持单比特量子门（X, Y, Z, H, S, T）操作
- 显示当前态矢量、密度矩阵、测量概率与布洛赫坐标

### 2. **多量子比特系统可视化**
- 同时展示多个量子比特的布洛赫球
- 支持多比特门（CNOT, SWAP, CZ）操作
- 可创建纠缠态（Bell态、GHZ态、W态）
- 显示张量积形式的全局态与纠缠度量

### 3. **量子算法演示**
- **Deutsch-Jozsa算法**：判断函数是否为常函数或平衡函数
- **Grover搜索算法**：在无序数据库中查找目标项
- **量子傅里叶变换 (QFT)**：实现量子态的频域转换
- **量子相位估计 (QPE)**：估计量子算子的本征相位

### 4. **量子计算模拟器**
- 支持自定义量子门序列（支持 X0, H1, CNOT01, RX2(45) 等格式）
- 实时计算并显示最终态矢量（Dirac记号与复数形式）
- 模拟测量过程，显示各基态概率分布
- 支持1至5个量子比特的模拟（保证性能）

---

## 🛠 技术栈

- **前端框架**：原生 JavaScript + HTML5 + CSS3
- **三维渲染**：Three.js（WebGL）
- **数学计算**：math.js（复数运算、线性代数）
- **交互控制**：OrbitControls（三维视角控制）
- **模块化**：使用IIFE封装各模块，避免全局污染

## 🧪 使用示例

### 单比特操作
1. 点击“单量子比特”选项卡
2. 使用滑块调整 θ 和 φ
3. 点击“H”按钮应用Hadamard门
4. 观察布洛赫球上状态向量的变化与右侧信息更新

### 多比特纠缠态创建
1. 切换到“多量子比特系统”选项卡
2. 选择“2个量子比特”
3. 点击“创建 Bell 态”
4. 观察两个量子比特的状态箭头与纠缠连接线

### 运行量子算法
1. 切换到“量子算法演示”选项卡
2. 选择“Grover 搜索”
3. 设置比特数 n=3，目标索引=5
4. 点击“运行算法”
5. 查看最终态分布与测量结果

## 📊 显示信息说明

- **态矢量**：Dirac记号与复数形式
- **布洛赫坐标**：(x, y, z) 单位球坐标
- **密度矩阵**：2×2复数矩阵（单比特）
- **测量概率**：各计算基态的概率百分比

---

## ⚠️ 注意事项

1. 多量子比特模拟受限于浏览器性能，建议不超过5个比特
2. 复数运算采用 math.js 库，可能存在浮点误差
3. 三维渲染需要支持 WebGL 的浏览器
4. 测量模拟为概率采样，结果具有随机性
